# DecodeLabs Internship Projects

This repository contains three data analytics projects completed as part of the DecodeLabs Internship Program.

## Project 1: Data Cleaning & Data Quality Assessment

* Performed data cleaning on raw e-commerce transactional data.
* Handled missing values, duplicate records, and inconsistent formatting.
* Standardized date and text fields.
* Validated numerical attributes and generated a cleaned dataset for further analysis.

### Tools Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

---

## Project 2: Exploratory Data Analysis (EDA)

* Conducted exploratory analysis on the cleaned dataset.
* Analyzed customer behavior, sales trends, payment methods, and product performance.
* Identified business patterns and generated actionable insights.
* Created visualizations to support data-driven decision making.

### Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## Project 3: E-Commerce SQL Analytics & Business Intelligence

* Imported e-commerce data into SQL Server.
* Performed business analysis using SQL queries.
* Analyzed revenue, customer behavior, product performance, payment methods, and referral sources.
* Generated business insights through data aggregation and reporting.

### Tools Used

* SQL Server Management Studio (SSMS)
* SQL
* Microsoft Excel
