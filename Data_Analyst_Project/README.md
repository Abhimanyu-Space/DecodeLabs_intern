# E-Commerce SQL Analytics & Business Intelligence

## Project Overview

The objective of this project was to perform business intelligence analysis using SQL Server Management Studio (SSMS). SQL queries were used to analyze sales performance, customer behavior, product performance, payment trends, referral sources, and order status.

## Business Questions Answered

* What is the total revenue generated?
* Which products generate the highest revenue?
* Which products have the highest sales volume?
* Who are the top customers?
* Which payment methods are preferred by customers?
* Which referral sources drive the most revenue?
* How does revenue change over time?
* What is the distribution of order statuses?

## SQL Concepts Used

* SELECT
* WHERE
* GROUP BY
* ORDER BY
* COUNT()
* SUM()
* AVG()
* DISTINCT
* Aggregate Functions

## Technologies Used

* SQL Server Management Studio (SSMS)
* SQL
* Microsoft Excel

## Key Outcomes

* Measured overall business performance
* Identified top revenue-generating products
* Evaluated customer purchasing patterns
* Analyzed payment method effectiveness
* Generated actionable business insights through SQL analysis
