# Exploratory Data Analysis (EDA)

## Project Overview

This project focuses on exploring the cleaned e-commerce dataset to understand customer behavior, product performance, sales trends, and operational patterns. The goal was to extract meaningful insights that could support business decision-making.

## Analysis Performed

* Dataset overview and summary statistics
* Customer purchasing behavior analysis
* Product performance analysis
* Payment method analysis
* Revenue trend analysis
* Order status analysis
* Outlier detection
* Business insight generation

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Key Outcomes

* Identified top-performing products
* Analyzed customer purchasing trends
* Evaluated revenue distribution
* Studied payment preferences
* Generated business insights through data visualization
