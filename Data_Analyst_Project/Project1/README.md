# Data Cleaning & Data Quality Assessment

## Project Overview

The objective of this project was to transform raw e-commerce data into a clean and analysis-ready dataset. Data quality issues such as missing values, duplicate records, inconsistent formatting, and incorrect data types were identified and resolved.

## Key Tasks Performed

* Imported and inspected raw dataset
* Handled missing values
* Removed duplicate records
* Standardized text formatting
* Converted date columns to proper datetime format
* Validated numerical fields
* Generated a cleaned dataset for further analysis

## Technologies Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

## Project Outcome

The final cleaned dataset was prepared for exploratory data analysis and business intelligence reporting while ensuring consistency, reliability, and improved data quality.
